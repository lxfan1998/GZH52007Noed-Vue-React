# 什么是微信公众账号？
指的是我们的微信app产品里面可以为广大的民众提供功能服务的一类产品。

+ 各类银行的公众账号
+ 中国联通
+ 中国移动
+ 商城里面的公众号
+ 吃饭消费、会员、积分

# 产品分类
+ 订阅号
    + 小林漫画
    + 糗事百科
    + ...
    
+ 服务号
    + 中国联通
    + 工商银行

+ 区别：
    + 在微信app里面展示的位置不一样
        + 订阅号：专门有一个订阅号的栏目存放所有关注的订阅号
        + 服务号和用户列表一栏

    + 消息推送
        + 服务号：每天可以推送一条信息
        + 服务号：一个月只能四次
    + 权限不一样，用户如果要自定义微信公众号服务开发，腾讯对不同类型的账号开发出来的权限是不一样的。
        + 订阅号：无法做微信支付
        + 服务号：可以做支付
        
# 可以做什么？
+ 信息的推送，吸引用户，获取流量（1. 坚持 2. 原创 3. 新意）
+ 提供服务
